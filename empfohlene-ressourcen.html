Recommended (Online-) Ressources for your side projects

By Fabio Maienschein
18/09/2020
Students, as future workers and decision makers, are an ideal target group for software tool vendors, so many vendors have started to provide generous free trial or free plans to convince them of the quality of these offerings. Such resources are ideal for smaller projects to build on and gain experience with these technologies. In this lesson you will learn about various offerings and resources that you can use for your next side project!

1) Free tools for design and development

The following free tools are especially popular within designers and developers:

Visual Studio Code: The open source code editor from Microsoft, which is a general-purpose weapon for development projects due to its simple, but powerful user interface and can be adapted for all kinds of use-cases with its rich plugin ecosystem.

Figma: A modern design tool that allows the design of user interface mockups and provides clickable prototypes that can be shared with a link. Figma also emphasizes collaborative features that facilitate a common design and review process. Check out this blog post to help you get started with Figma.

Git: Git has become the most popular versioning and source code management tool in recent years and is essential for all coding projects. Hosted source control platforms like GitHub und GitLab allow you to store source code publicly (or privately), and to host it directly (GitHub Pages) or as the starting point for Continuous Integration/ Continuous Deployment (GitHub Actions).
The HM also offers its students a self-hosted GitLab instance, that all students with HM credentials (@hm.edu) can use.

2) Free offers for hosting and cloud infrastructure

The following providers are particularly recommended due to their generous free plans for hosting smaller projects and websites:

AWS Educate - Student access to Amazon's cloud services with $100 starting credit (no credit card required). Follow this blog post to claim your own AWS resources.

Netlify - extremely simple deployments of front-end resources (websites and applications) directly from the source code management or alternatively via drag & drop of local files

Firebase - Backend-as-a-Service service from Google that provides free quotas for services such as authentication, hosting and storage (Firestore).

Heroku - Hosting of web applications (free plan with restrictions, option of a free hobby dyno via GitHub Student)

Amazon Webservices, Microsoft Azure, Google Cloud Platform - even the large business clouds offer free plans, often without a credit card (e.g. Azure Student).

3) GitHub Student Developer Pack

Probably the largest collection of free software resources, the GitHub Student Developer Pack deserves a mention in any such list. Students with valid university addresses will receive a variety of offers after verification by GitHub. These include free domains (.me and .tech, limited to 1 year), cloud resources (AWS, Azure, Heroku), a cooperation with Unity (free assets), professional development environments for a variety of programming languages (Jetbrains) sas well as one to six month accesses for Tech course platforms (FrontendMasters, One Month, Flatiron School).
 
A complete overview of all included freebies can be found here, also it's worth taking a look into the gallery of projects built with resources from the GitHub Student Pack.
