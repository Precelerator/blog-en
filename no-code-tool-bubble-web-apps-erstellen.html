 No-Code Tool Bubble: Visually create web apps

By Fabio Maienschein
18/09/2020

Whether TripAdvisor, AirBnB or Shopify: Web-apps represent the most important platform for consumer and B2B applications alongside mobile apps. Until a few years ago, this area was still dependent on complex web development, but more and more so-called no- or low-code platforms are emerging which allow even less technically skilled users to create web applications.

One of these no-code platforms is Bubble, a young start-up from New York. The platform promises to simplify the creation of fully functional web applications using the company's own visual programming language. Web applications created using Bubble are hosted by Bubble itself and are accessible to users via regular web URLs.

Features

Bubble offers various features such as authentication (logon/registration), storage, retrieval and processing of data, connection to APIs (like interfaces to any backend) and the connection of payment providers (e.g. Stripe). Created web applications can be built responsive, i.e. suitable for all screen sizes, so that they can also be displayed appropriately on smartphones and tablets.

Principles of Bubble

The user interface of bubble applications can be assembled from a library of interface components (buttons, text fields, etc.) by drag and drop. So-called workflows regulate the processes that should be executed by the application during user interactions. 

Learn how to use Bubble

Bubble offers a lot of resources to help you get started. Go to Bubble.io to create your own account and get started. There's 10 quick tutorials that show you step by step to create little applications such as a shopping list, iTunes song search and an image gallery that users can upload images to.

If you want to work on a Bubble app as a team, go to Settings -> General -> Application rights and change the dropdown from "Private" to "Everyone can edit". Now if you copy the link of your Bubble app (while it's open in the editor), you can share it with your team so you can all edit the same application!
